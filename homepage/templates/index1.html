ArmorFS

A cryptpgraphic filesystem in userspace. The filesystem encrypts and decrypts the user's data as per requirements. Userspace ensures that it is usable by each user (even non-root) and linked to each individual.
Created for the final year project, with sponsorship from Veritas Technologies LLC.

  • Cryptography
  • Filesystem
  • Userspace
Diabetes Predictor

A website utilizing machine learning algorithm to predict the diabetes condition of an individual based on different parameters.
Used the Pima Indian's dataset for the ML algorithm and Flask framework to create the website.

  • Machine learning
  • Flask
  • Website

CHMS
